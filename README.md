# blockalign README

Blockalign will align blocks of code automatically.

## Features

Align blocks of code. No need to select the block, it will auto-detect possible alignments. 

## Suggested keymap

[
   { "key": "ctrl+\\",           "command": "extension.blockalign",
                                 "when": "editorTextFocus" },
]
